What this measures

  • Decode Time: Time for AudioContext.decodeAudioData() to fully decode the file into a PCM AudioBuffer.
  • PCM Buffer Size: Uncompressed memory footprint = channels × samples × 4 bytes (Float32).
  • Seek Latency: Average time to create a BufferSourceNode and start playback from 10 random offsets (measures seek overhead, not network).
  • Each test is independent — previous buffers are released before the next load.
